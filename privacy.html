Privacy policy

How files and site data are handled

This page explains the hosted browser workflow, downloadable scripts, and the site policies needed for a trustworthy utility website.

Home / Privacy

Hosted conversion processing

The browser tool uploads PDFs to a Python backend so the server can run the same `pdf_to_epub.py` and `format_epub.py` pipeline used in the downloadable workflow.

Uploaded files should be treated as temporary processing inputs. Production deployment should include clear retention limits, cleanup rules, and host-level security controls.

Downloadable scripts

If you use the Python scripts, processing happens on your own machine in your local environment. The site provides the files and instructions, but the conversion itself runs where you execute the script.

Third-party libraries

The hosted site may use frontend libraries for UI behavior and a Python backend for conversion work. Deployment should document any production libraries, infrastructure, or third-party services actually used.

Advertising and analytics

This site uses Google Analytics (GA4) to collect anonymous usage data such as page views, session duration, and general traffic patterns. Google Analytics uses cookies and may collect your IP address. No personally identifiable information is collected or shared through analytics.

For more information on how Google handles analytics data, see the Google Privacy Policy. You can opt out of Google Analytics tracking by installing the Google Analytics Opt-out Browser Add-on.

Content and external links

Guides on the site may link to external tools, references, or documentation. External sites have their own privacy practices and should be reviewed separately.

Updates

This policy will be revised as the site grows, particularly before enabling AdSense, newsletter forms, or other user-facing integrations.

Quick privacy summary

  • Browser uploads are processed by a Python backend
  • Python scripts can still run locally if downloaded
  • Google Analytics collects anonymous usage data
  • Future ads or other integrations will be disclosed here
